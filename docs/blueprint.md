# **App Name**: Dulus Business Hub

## Core Features:

- Dashboard Metrics: Display key business metrics (revenue, clients, invoices, tasks) using clear and engaging Stat Cards.
- Client Management: Manage client information (name, email, phone) with the ability to add, view, edit, and delete records.
- Invoice Management: Create, edit, view, and delete invoices, tracking payment statuses (Paid, Unpaid, Overdue) automatically.
- Task Management: Add tasks with descriptions, due dates, and priority levels, and mark tasks as completed.
- Inventory Tracking: Manage items with SKU, name, quantity, and price, adding, updating, and removing inventory items easily.
- Revenue Insights: Use a tool to Generate reports summarizing financial and client revenue and outstanding payments, to offer actionable business insights. The tool decides whether to generate the insight or return a message saying that it is unable to provide an insight, depending on data availability
- User Authentication: Secure user authentication with email/password.

## Style Guidelines:

- Primary color: Use the specified Blue (#459BFF) as the primary color to convey professionalism, trust, and cleanliness, as it's in the original company image.
- Background color: Very light desaturated blue (#E0F0FF). The desaturation is above 10% to keep the design aligned to professional design standards. The brightness is very high, befitting a light scheme.
- Accent color: Shifted about 30 degrees toward green-blue in hue from the primary to the blue (#45BF9B); lower saturation than the primary creates visual interest.
- Font: 'Inter' (sans-serif) for both headlines and body text, given that this app does not have literary or high-fashion qualities that would call for more expressive or specific choices. Note: currently only Google Fonts are supported.
- Persistent left sidebar navigation with a top navbar for profile and settings, ensuring efficient access to all tools. Rounded components should give a modern feel.
- Use subtle transitions for fluid user interactions. Ensure components follow accessibility best practices.
